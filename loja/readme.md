# Projeto Loja:
 - Python v 3.8.3;
 - Django 3.1.7;
 - Django Rest Framework 3.12.2;
 - Markdown 3.3.4;
 - Django Filter 2.4.0.
 - Banco de Dados SQLite;
 
## Execução do projeto
Dentro da pasta loja.
### Instalar dependências:
pip install  -r requirements.txt
### Executar Migrações:
python manage.py migrate
### Subir servidor local:
python manage.py runserver 
Subirá o servidor localhost na porta 8000.

OBS: O projeto Loja conta com um App produto, o recurso está disponível no Endpoint: http://localhost:8000/produtos/
